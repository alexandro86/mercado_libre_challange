# mercado_libre_challange
Proyecto challange busqueda mercado libre

#Instrucciones
## Descargar proyecto
$ git clone https://github.com/alexandro86/mercado_libre_challange.git

Desde la raíz del proyecto
$ cd mercado_libre_challange
$ cd client
$ npm i
$ yarn start

Desde la raíz del proyecto
$ cd server
$ npm i
$ npm run start
